## 🚀 Overview

An interactive **Decision Tree Classification Dashboard** built with Streamlit that demonstrates how decision trees classify data using feature-based splits.

The app visualizes **decision boundaries, model performance, and tree structure**, helping users understand how the algorithm works internally.

---

## ✨ Features

* 🌳 Train a Decision Tree classifier
* 🎛️ Adjustable hyperparameters (criterion, max depth)
* 📊 Accuracy & confusion matrix
* 🗺️ Decision boundary visualization
* 🌿 Tree structure visualization
* 🎯 Real-time prediction tool
* 👀 Dataset preview

  * Purchase Decision (0 / 1)

## 📈 Visualizations

* 🗺️ Decision boundary (Training & Test sets)
* 📊 Confusion matrix
* 🌿 Tree structure diagram
* 📈 Accuracy metric

---

## 🧠 How It Works

1. Dataset is loaded from CSV
2. Data is split into training and test sets
3. Features are scaled
4. Decision Tree model is trained
5. Predictions and visualizations are generated

---

## 🎯 Live Prediction

Users can input:

* Age
* Salary

The model predicts whether the user is likely to **purchase or not**.
